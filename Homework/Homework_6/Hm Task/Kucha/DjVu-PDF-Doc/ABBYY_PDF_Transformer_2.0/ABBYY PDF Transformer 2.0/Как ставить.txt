Инструкция по установке.

1. Отключить интернет, отключить антивирус. 

2. Установить с ехе-файла.

3. Распаковать лекарство.

4. Вставить распакованный файл лекарства в папку Program files / ABBYY PDF Transformer 2.0

5. Запустить лекарство, при запуске открывайте эту же папку, выбирайте ABBYY PDF Transformer 2.0.exe
(я делаю так, хотя наверное неважно какой файл выбрать, у меня уже программа стоит и сносить ее не хочется).

И не обращайте внимания на запись "done". Да черт с ним, "done", так "done". Просто теперь закройте лекарство, оно свою функцию уже выполнило.

6. А теперь попробуйте открыть программу, например что-нибудь создать в формате PDF. У вас не должно быть записи про 15 дней.
И только теперь можно включать интернет!

Это главная ошибка пользователей, они пытаются установить программу с включенным интернетом.

А программа при установке отсылает запрос на родной сайт и дает вам только испытательную версию. Вы начинаете искать новые серийные номера, патчи, кряки, ключи.
А на самом деле все работает нормально, данная версия программы у вас уже нормальная и постоянная.
Если Вы уже все сделали неправильно и получили испытательную версию, то выход следующий.
1. Деинсталировать программу, причем после деинсталяции удалите даже саму папку (Program files / ABBYY PDF Transformer 2.0), потому что удаления там остались файлы с временной лицензией. Вообщем, папку удалить тоже. 2. Перезагрузить компьютер. 3. Не включать Интернет. 4. Установить заново, как написано в Инструкции выше.

Описание: ABBYY PDF Transformer 2.0 – универсальный инструмент для работы с PDF-файлами.
Он не только преобразует PDF-файлы в привычные, удобные для редактирования форматы – Microsoft Word, Excel, HTML и TXT, но также позволяет создавать PDF прямо из приложений Microsoft Office .
Кроме того, программа позволяет создавать PDF-документы практически из любого приложения, поддерживающего вывод документа на печать (это делается при помощи драйвера принтера ABBYY PDF Transformer 2.0). 

ABBYY PDF Transformer быстро и точно конвертирует PDF-файлы, сохраняя оформление исходного документа — таблицы, картинки, расположение абзацев и т.д.
Для создания PDF-файлов из документов Microsoft Word, Excel, PowerPoint или Visio на компьютере должно быть установлено соответствующее приложение Microsoft Office. 

Поддерживаемые форматы:
Вы можете конвертировать PDF в форматы: Microsoft Word 2003/XP/2000 (DOC), Microsoft Excel 2003/XP/2000 (XLS), Rich Text Format (RTF), PDF с возможностью полнотекстового поиска, TXT, HTML 4.0
Вы можете создавать PDF из файлов: Microsoft Word (DOC/RTF), Microsoft Excel (XLS), Microsoft PowerPoint (PPT), Microsoft Visio (VSD) 

Используйте виртуальный принтер PDF-XChange for ABBYY PDF Transformer 2.0 для
создания PDF-документов из любых приложений, поддерживающих вывод документов на
печать (через меню Файл> Печать).
Можно конвертировать PDF-документы и создавать PDF на практически любых широко используемых языках мира.
